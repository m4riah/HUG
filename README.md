# HUG+
A HUG+ é um projeto que visa facilitar o acesso a atendimentos psicoterapêuticos remotos, de maneira simples e descomplicada nos formatos particular e social. O foco é criar espaços sustentáveis especialmente para pessoas de baixa renda, utilizando a tecnologia para tornar os serviços de saúde mental mais acessíveis e inclusivos.
